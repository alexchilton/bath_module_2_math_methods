"Question 3 [6]: Consider the following formula of propositional logic:\n",
"**P = ((A ∨ B) ∧ C → (A ∧ B) ∨ C)**\n",
"\n",
"a) Assume that A is true, and B,C are false. Is P true or false? Briefly explain why.\n",
"\n",
"b) Which truth values for A,B,C will result in P being false?"
"a) **A is true, B and C are false**\n",
"\n",
"Given: A = T, B = F, C = F\n",
"\n",
"A ∨ B = T ∨ F = T\n",
"\n",
"(A ∨ B) ∧ C = T ∧ F = F\n",
"\n",
"A ∧ B = T ∧ F = F\n",
"\n",
"(A ∧ B) ∨ C = F ∨ F = F\n",
"\n",
"P = ((A ∨ B) ∧ C) → ((A ∧ B) ∨ C) = F → F = T\n",
"\n",
"**P is true.**\n",
"\n",
"This is because a conditional statement (→) is true whenever the antecedent is false, regardless of the truth value of the consequent. Since ((A ∨ B) ∧ C) evaluates to false, the entire implication is true."
